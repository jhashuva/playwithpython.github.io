OBJECT ORIENTED CONCEPTS

Class

In python creating class with the help of a keyword class.

class <className>:
    pass

Example 1:

class Student:
    pass

Example 2: 

class Circle:
    pass

You have seen how to create a class. Now let us see what are the methods we need to declare in the class.

  • init method __init__() : It is similar to constructors in C++ or JAVA. Constructors are used to initialize the objects state. The work of the constructors are used to initialize the object state and initialize values to the data members of the class when an object of class is created. Lets see implementation of __init__() on the above mentioned classes.

Example 1:

class Student:
    def __init__(self, rollnum, full_name, dob):
        self.rollnum = rollnum
        self.full_name = full_name
        self.dob = dob

Example 2:

class Circle:
    def __init__(self,radius):
        self.radius = radius

self keyword is used to represent instance of the class. It binds the attributes with given attributes. - what happenes if you ignore or forgot to mention self. Python thows an error.

  • is self is a keyword ? : You could use any other word for example look the below example.
class ExampleClass:
    def __init__(this,a,b):
        this.a = a
        this.b = b

Lets try to implement other methods for the classes in above mentioned examples.

class Student:
    def __init__(self,full_name,roll_no,dob):
        self.full_name = full_name
        self.roll_no = roll_no
        self.dob = dob
    #this method will split the name into first name and last name
    def name_split(self):
        return self.full_name.split()
    #this method will generate email and return it.
    def get_mail(self):
        return self.full_name+'@company.com'

    def get_yob(self):
        dob = self.dob.split('/')
        return dob[2]

    def get_mob(self):
        dob = self.dob.split()
        return dob[1]

    def get_dob(self):
        dob = self.dob.split('/')
        return dob[0]

    def set_dob(self,dob):
        self.dob = dob
